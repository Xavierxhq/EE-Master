# excellent-education

## Build Setup

#### 将以下代码中的 npm 替换成 cnpm(前提是你的电脑已经安装了npm并配置了淘宝镜像)
#### 这里使用到前两句就够了，会自动打开浏览器的
``` bash
# 安装依赖
npm install

# 运行项目（已配置热加载）
npm run dev

# 生成（打包）项目
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
